# Vowel Planning

For the first lesson the user should learn 2 vowels
Each vowel follows the following learning pattern:

- [x] Learn vowel (both times if multiple ways of writing)
- [x] Match vowel with sound
- [x] Match sound with vowel
- [x] Match vowel with sound (combined with consonant)
- [x] Match sound with vowel (combined with consonant)
- [x] Learn vowels again just to review before matching game
- [x] Two sets of matching game with the vowels that are combined with consonants,
  and the ones that are not
- [x] Randomized for each of the 2 vowels:
  - [x] Match vowel with sound
  - [x] Match sound with vowel
  - [x] 2x Match vowel with sound (combined with consonant)
  - [x] 2x Match sound with vowel (combined with consonant)
- [x] Two sets of matching game with the vowels that are combined with consonants,
  and the ones that are not
